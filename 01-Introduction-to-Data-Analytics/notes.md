# 01 · Introduction to Data Analytics

## What is Data Analytics?
Data analytics is the process of examining raw data to find trends, answer questions, and support business decisions.

## Types of Data Analytics
1. **Descriptive** — What happened? (reports, dashboards)
2. **Diagnostic** — Why did it happen? (root cause analysis)
3. **Predictive** — What will happen? (forecasting, ML)
4. **Prescriptive** — What should we do? (recommendations)

## Data Analyst Workflow
1. Ask the right business question
2. Collect / access data
3. Clean & prepare data
4. Analyze (EDA, stats)
5. Visualize
6. Communicate insights & recommendations

## Key Skills to Build (covered in this repo)
- Excel, SQL — data extraction & manipulation
- Python (Pandas, NumPy) — analysis at scale
- Matplotlib, Seaborn, Power BI — visualization & dashboards
- Statistics — sound reasoning about data
