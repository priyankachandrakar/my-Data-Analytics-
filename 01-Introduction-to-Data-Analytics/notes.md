## What is Data Analytics?
What is Data Analytics?

Data Analytics is the process of collecting, cleaning, analyzing, and
interpreting data to discover useful insights, identify patterns, and
support better decision-making.

In simple terms, Data Analytics transforms raw data into meaningful
information that businesses and organizations can use to improve
performance and solve problems.

Data Analytics Process

1.  Collect Data
2.  Clean Data
3.  Process Data
4.  Analyze Data
5.  Visualize Results
6.  Make Data-Driven Decisions

Why is Data Analytics Important?

-   Helps businesses make informed decisions.
-   Identifies trends and patterns.
-   Improves operational efficiency.
-   Enhances customer satisfaction.
-   Supports business growth and profitability.

Real-World Examples

-   Amazon recommends products based on your shopping history.
-   Netflix suggests movies and TV shows based on your viewing
    preferences.
-   Google Maps analyzes traffic data to recommend the fastest route.
-   Hospitals use data analytics to improve patient care.
-   Banks detect fraudulent transactions using data analysis.

## Types of Data Analytics
Types of Data Analytics

Data Analytics is generally divided into four main types. Each type
answers a different business question and helps organizations make
better decisions.

1. Descriptive Analytics

Question Answered: What happened?

Descriptive Analytics summarizes historical data to understand past
performance.

Examples: - Monthly sales reports - Website traffic summary - Total
revenue by year

------------------------------------------------------------------------

2. Diagnostic Analytics

Question Answered: Why did it happen?

Diagnostic Analytics identifies the reasons behind trends or problems by
comparing and exploring data.

Examples: - Why did sales decrease? - Why did customer satisfaction
drop? - Why did website traffic increase?

------------------------------------------------------------------------

3. Predictive Analytics

Question Answered: What is likely to happen next?

Predictive Analytics uses historical data and statistical models to
forecast future outcomes.

Examples: - Sales forecasting - Customer churn prediction - Demand
forecasting

------------------------------------------------------------------------

4. Prescriptive Analytics

Question Answered: What should we do?

Prescriptive Analytics recommends the best actions based on predictions
and business goals.

Examples: - Product recommendations - Dynamic pricing - Inventory
optimization

------------------------------------------------------------------------

Summary Table

1.  Descriptive Analytics – What happened?
2.  Diagnostic Analytics – Why did it happen?
3.  Predictive Analytics – What will happen?
4.  Prescriptive Analytics – What should we do?

Key Takeaway

The four types of Data Analytics work together to understand the past,
explain the present, predict the future, and recommend the best course
of action.


## Data Analyst Workflow
Data Analytics Lifecycle

What is the Data Analytics Lifecycle?

The Data Analytics Lifecycle is a structured, step-by-step process used
to transform raw data into meaningful insights that support better
business decisions.

Step 1: Data Collection

Data is gathered from multiple sources such as: - Databases - Websites -
APIs - Surveys - Excel files - IoT devices

Goal: Collect accurate and relevant data.

------------------------------------------------------------------------

Step 2: Data Cleaning

Raw data often contains errors. In this step: - Remove duplicate
records - Handle missing values - Correct incorrect data - Standardize
formats

Goal: Improve data quality.

------------------------------------------------------------------------

Step 3: Data Processing

The cleaned data is transformed into a format suitable for analysis.
Common tasks include: - Merging datasets - Filtering records - Creating
new columns - Feature engineering

Goal: Prepare data for analysis.

------------------------------------------------------------------------

Step 4: Data Analysis

The prepared data is analyzed using tools such as: - Excel - SQL -
Python - Power BI

The objective is to identify patterns, trends, and relationships.

------------------------------------------------------------------------

Step 5: Data Visualization

Results are presented using: - Charts - Graphs - Dashboards - Reports

Visualization makes insights easier to understand.

------------------------------------------------------------------------

Step 6: Decision Making

Business stakeholders use the insights to: - Improve products - Increase
sales - Reduce costs - Solve business problems - Make data-driven
decisions

------------------------------------------------------------------------

Summary

1.  Collect Data
2.  Clean Data
3.  Process Data
4.  Analyze Data
5.  Visualize Results
6.  Make Data-Driven Decisions

Key Takeaway

The Data Analytics Lifecycle ensures that data is collected, prepared,
analyzed, and presented in a systematic way to generate valuable
insights and support effective decision-making.

======================================== Data Analyst vs Data
Scientist ========================================

Data Analyst

A Data Analyst collects, cleans, analyzes, and visualizes data to help
organizations make informed business decisions.

Responsibilities

-   Analyze historical data
-   Create reports and dashboards
-   Identify trends and patterns
-   Support business decisions

Skills

-   Excel
-   SQL
-   Python
-   Power BI
-   Statistics

------------------------------------------------------------------------

Data Scientist

A Data Scientist uses advanced statistics, machine learning, and
programming to predict future outcomes and build intelligent models.

Responsibilities

-   Build predictive models
-   Develop machine learning algorithms
-   Analyze large datasets
-   Solve complex business problems

Skills

-   Python
-   Machine Learning
-   Statistics
-   SQL
-   Deep Learning

Difference

Data Analyst: - Focuses on past and present data - Creates reports and
dashboards - Answers “What happened?”

Data Scientist: - Focuses on future predictions - Builds AI and ML
models - Answers “What will happen?”
