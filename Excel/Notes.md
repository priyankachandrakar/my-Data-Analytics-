Excel Learning Notes (Beginner to Advanced)

============================================================ 01.
Introduction to Microsoft Excel
============================================================

Microsoft Excel is a spreadsheet application developed by Microsoft. It
is widely used for data entry, calculations, reporting, data analysis,
dashboard creation, and business decision-making.

Why Learn Excel?

-   Organize data efficiently
-   Perform calculations
-   Analyze large datasets
-   Create charts and dashboards
-   Essential skill for Data Analysts

  ---------------------
  02. Excel Interface
  ---------------------

Main Components: 1. Title Bar 2. Quick Access Toolbar 3. Ribbon 4. Name
Box 5. Formula Bar 6. Worksheet 7. Rows 8. Columns 9. Cells 10. Sheet
Tabs 11. Status Bar 12. Zoom Slider

  ----------------
  03. Data Types
  ----------------

Excel supports different types of data:

• Text • Number • Currency • Percentage • Date • Time • Boolean
(TRUE/FALSE) • Formula

Example: Name Salary John 50000 Priya 65000

  ---------------------
  04. Cell References
  ---------------------

Relative Reference Example: =A1+B1 Changes automatically when copied.

Absolute Reference Example: =$A1+B$1 Never changes when copied.

Mixed Reference Example: =$A1 or A$1

  --------------------
  05. Basic Formulas
  --------------------

SUM() → Adds numbers AVERAGE() → Calculates average COUNT() → Counts
numeric cells COUNTA() → Counts non-empty cells MAX() → Largest value
MIN() → Smallest value ROUND() → Rounds a number

Example: =SUM(A1:A10) =AVERAGE(B2:B20)

  ------------------------------
  12. XLOOKUP vs INDEX + MATCH
  ------------------------------

XLOOKUP • Easier to use • Replaces VLOOKUP • Searches left or right

INDEX + MATCH • More flexible • Faster for large datasets • Preferred in
advanced Excel
